Graph-Ex

Introductions:

This project was created as a term project for DSA 2021. Tajammul Naeem and I (Nabeel Raza) collaborated to create something that we have not worked on previously. We both are students at the prestigious institution of University of Engineering & Technology, Lahore majoring in Computer Engineering. We wish to create more useful tools for our society and make coding a culture in Pakistan.

Motivations:

Some students were observed to have difficulty understanding how graphs are plotted so this utility is to help them. Moreover, we wanted to try and explore the field of Artificial Intelligence. So we decided to create this project.

Learning Outcomes:

This project introduced us to Machine Learning and Deep Learning concepts. We used Kaggle to create our model. Furthermore, we used Django to learn how websites are created. We learned all this in the three months of time span. We ought to practice more but due to strict time deadlines and continuous headaches (due to DSA assignments) we had to make do with what little we could learn. More things, like scaling the graph, better usage of equations can be expected.
